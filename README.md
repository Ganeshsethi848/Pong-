# Pong Game Documentation

## Overview
Pong is a two-dimensional sports game that simulates table tennis. Players control paddles by dragging them vertically across the screen's left and right sides. The goal is to strike the ball back and forth and score points by getting the ball past the opponent's paddle.

## Features
- Single-player mode against AI
- Two-player mode
- Adjustable paddle size
- Score tracking
- Simple yet engaging gameplay

## Controls
- **Player 1:** Use the up and down arrow keys to move the left paddle.
- **Player 2:** Use the 'W' and 'S' keys to move the right paddle.

## Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/Ganeshsethi848/Pong-
   ```
2. Navigate to the project directory:
   ```bash
   cd Pong-
   ```
3. Open the HTML file in a web browser to start playing.
